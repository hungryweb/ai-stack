# Hungryweb AI Stack

This repository contains the Hungryweb AI stack.

## Tools

This stack will document tested AI development tools, workflows, and setup notes as they are evaluated.

- **Cursor**: AI-powered IDE used for code navigation, editing, refactoring, planning, and agent-based development inside the project.
- **Claude Code**: Terminal-based coding assistant for repository-level changes, debugging, automation, and command-line workflows.
- **Codex**: AI coding agent used for implementation support, code review, test generation, and experiments with software development tasks.

New tested tech stack details, recommendations, and usage notes will be published here.
